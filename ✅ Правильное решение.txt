Как подключиться к проекту
1️⃣ Скачать проект

В терминале выполнить:

git clone https://github.com/nobilauto-it/Admin-Dashboard.git


Зайти в папку проекта:

cd Admin-Dashboard

2️⃣ Открыть проект в Cursor

Открыть Cursor →
File → Open Folder → выбрать папку Admin-Dashboard

3️⃣ Переключиться на ветку dev

В терминале внутри Cursor:

git checkout dev


Если ветка не подтянулась:

git fetch origin
git checkout dev

4️⃣ Установить зависимости (первый запуск)

Если Python проект:

pip install -r requirements.txt


Если есть фронт:

npm install

5️⃣ Работать правильно

Все изменения делать в ветке dev

Commit → Push

В main не работать

Проверить текущую ветку:

git branch


Звёздочка * должна стоять возле dev.

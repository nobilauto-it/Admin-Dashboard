--------------------------------------------------
Конфигурация внешнего API
--------------------------------------------------

Базовый URL:
    http://194.33.40.197:7070

Известные эндпоинты, которые сейчас использует приложение (на 2026-01-28):
    - /api/data/deals
        Назначение: загрузка данных по сделкам для страницы Deals.

    - /api/entity-fields/?type=deal
        Назначение: получить доступные поля для сущности "сделка" (deal).

    - /api/entity-data/?type=deal
        Назначение: получить записи по сделкам (универсальный эндпоинт entity-data).

    - /api/entity-fields/?type=smart_process&entity_key=sp:1114
        Назначение: получить поля для смарт-процесса с ключом sp:1114.

    - /api/entity-data/?type=smart_process&entity_key=sp:1114
        Назначение: получить данные смарт-процесса с ключом sp:1114.

    - /api/entity-fields/?type=contact
        Назначение: получить поля для контактов.

    - /api/entity-data/?type=contact
        Назначение: получить данные по контактам.

    - /api/entity-fields/?type=lead
        Назначение: получить поля для лидов.

    - /api/entity-data/?type=lead
        Назначение: получить данные по лидам.

    - /api/processes-deals/
        Назначение: получить список воронок/процессов по сделкам.
